### Contribution
Hello reader,  
this is open source project. As it's meant for .NET developers, any interesant should be able to handle more or less complicated issues and contibute.

Eager to contribute?
There are two ways to help:
- Test the app and create an issue if either bug found or got an idea for improvement.
- Take an issue and fix/implement if it's well descripted, discussed and approved.

Coding style
- tabs over spaces (set any size suits you)
- specific types over var
